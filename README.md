# PyKids: Learn Python, Build the Future 🐍🚀

Welcome to **PyKids**! This is a vibrant, interactive, and kid-friendly website built to introduce 12-year-olds (and beginners of all ages!) to the magical world of Python programming.

## 🌟 Features
This project is built using modern web technologies (**React.js** and **Vite**) with a gorgeous, engaging user interface designed specifically for kids.

The curriculum includes **5 core lessons + 1 final challenge**:
1. **Intro**: What is programming? A look at the world with and without code.
2. **Variables (Lesson 1)**: Learning how to store data in "magic boxes".
3. **Logic (Lesson 2)**: Using `if/else` statements in a Secret Door Mini-Game!
4. **Math & Strings (Lesson 3)**: A magic potion calculator to mix numbers and words.
5. **Lists (Lesson 4)**: A digital backpack inventory manager.
6. **Functions (Lesson 5)**: A robot factory where kids build their own mini-robot.
7. **Madlibs Project**: Put it all together to create a funny Python story generator.
8. **Final Exercises**: A challenge page. Master it to get the Golden Python Award!

## 🛠️ Tech Stack
- **Framework**: React 18
- **Bundler**: Vite
- **Styling**: Vanilla CSS with modern Glassmorphism & custom variables
- **Icons**: Lucide React
- **Routing**: React Router DOM (v6)

## 🚀 Running the Project Locally

To run this project on your own machine, you will need [Node.js](https://nodejs.org/) installed.

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   cd KidPythonTeaching
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the development server**:
   ```bash
   npm run dev
   ```
4. Open your browser and navigate to `http://localhost:5173`.
